# Installation

```
gem build todo
gem install --local todo
```

# Usage

```
$ todo --help
Usage: todo [options]
    -i, --init                       initialize todo list file
    -t, --top                        give the top priority
    -a, --add                        add something to todo list
    -p, --pop                        remove top item from todo list
    -s, --sort                       interactively sort todo items by priority
    -h, --help                       Prints this help
```
